# Weapons

Five weapon types:

    - RifleAsalto
    - Francotirador
    - Escopeta
    - Pistola
    - Ametralladora Ligera

based off some from Fallout 2

## List of weapons

### Unordered

1. `RifleAsalto`: 
    * `FN FAL`
        + Variants
            - Default
            - Nightsight

        + Usable Ammo
            - 7.62mm
    * `AK-112`
        + Variants
            - Expanded Magazine
        + Usable Ammo
            - 5mm AP (Armor Piercing)
            - 5mm JHP (Jacketed Hollow Point)
2. `Francotirador`:
    * `M72 Gauss Rifle`

            - Extended Capacity
        + Usable Ammo
            - Microfusion Cell
3. `Escopeta`: 
    + `H&K CAWS`
        + Usable Ammo
            - 12 Gauge
    * `Pancor Jackhammer`
        + Usable Ammo
            - 12 Gauge
4. `Pistola`: 
    * `Desert Eagle .44`
        + Variants
            - Default
            - Expanded Magazine
        + Usable Ammo

            - .44 Magnum JHP (Jacketed Hollow Point)
    * `PPK12 Gauss Pistol`
        + Usable Ammo
            - 2mm EC (Electromagnetic Cartdrige)
    * `9mm Mauser`
        + Variants
            - Default
