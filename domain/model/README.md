# Weapons

Five weapons:

    - RifleAsalto
    - Francotirador
    - Escopeta
    - Pistola
    - Ametralladora Ligera

based off some from Fallout 2

## List of weapons

### Unordered

1. `RifleAsalto`: FN FAL
    + Variants
        - Default
        - Nightsight
        - HPFA (High Power Full Automatic)
    + Usable Ammo
        - 7.62mm
2. `Francotirador`: M72 Gauss Rifle
    + Usable Ammo
        - 2mm EC
3. `Escopeta`: H&K CAWS
    + Usable Ammo
        - 12 Gauge
4. `Pistola`: Desert Eagle .44
    + Variants
        - Default
        - Expanded Magazine
    + Usable Ammo
        - .44 Magnum FMJ (Full Metal Jacket)
        - .44 Magnum JHP (Jacketed Hollow Point)
5. `AmetralladoraLigera`: Light Support Weapon
    + Usable Ammo
        - 5mm AP (Armor Piercing)
        - 5mm JHP (Jacketed Hollow Point)
